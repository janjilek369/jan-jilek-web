## 3. O MNĚ

### Nadpis sekce
**Kdo jsem**

### Text (tykání — osobní tón)

Jmenuju se Honza Jílek a posledních 15 let dělám něco, co většinou dělají tři různí lidé: prodávám, školím a tvořím obsah. Všechno najednou.

Začínal jsem jako obchodní zástupce v oblasti Apple technologií pro školy. Postupně jsem si vybudoval roli, která daleko přesahuje klasický sales. Dnes pomáhám tisícovce škol s výběrem a nákupem technologií, ale i s jejich zavedením do praxe. Správa zařízení, školení učitelů, tvorba vzdělávacích materiálů.

Moje práce mi dává smysl, když vidím výsledek. Když z nápadu vznikne hotový projekt: web, webinář, newsletter, školení. Nepotřebuju tým deseti lidí. Potřebuju nápad, nástroje a prostor to dotáhnout.

Aktivně pracuju s AI nástroji. Ne jako s módním trendem, ale jako s každodenním pracovním nástrojem. Píšu o tom na Substacku, školím učitele a hledám způsoby, jak AI reálně šetří čas.

Mimo práci mě najdete v CrossFit boxu. Mám dvě děti a ženu, která se mnou naštěstí tu cestu sdílí.

### Poznámky pro design
- Foto vedle textu (jiné než v hero)
- Oddělený box nebo jemné pozadí
- Možnost rozbalit/zavřít delší text na mobilu

## TECHNICKÉ POZNÁMKY PRO CURSOR

### Stack
- **Framework:** React (nebo čistý HTML/CSS/JS — podle preference)
- **Styling:** Tailwind CSS
- **Deploy:** Vercel + GitHub
- **Fonty:** Inter nebo podobný moderní sans-serif
- **Barvy:** Neutrální základ (bílá/šedá/černá), jeden akcentní tón (modrá nebo tmavě zelená)

### Požadavky
- Single page s smooth scroll navigací
- Plně responzivní (mobile-first)
- Rychlé načítání (optimalizace obrázků)
- SEO základy (meta tagy, title, description)
- Open Graph tagy pro sdílení na LinkedIn
- Lazy loading pro obrázky
- Žádné zmínky o současném zaměstnavateli (24U)
- Web musí fungovat i za 2 roky bez úprav

### Struktura souborů (návrh)
```
/
├── index.html (nebo App.jsx)
├── styles/ (Tailwind config)
├── assets/
│   ├── images/ (fotky Jana, screenshoty projektů)
│   └── icons/
└── README.md
```
